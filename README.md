# PRO-CHISTKA Business Dashboard

Веб-приложение для управленческого контроля клинингового бизнеса PRO-CHISTKA.

Первая версия содержит:

- главный дашборд с ключевыми показателями;
- заявки;
- заказы;
- расходы;
- настройки бизнеса;
- демо-данные, совпадающие по смыслу с управленческой таблицей;
- подготовку к подключению Google Sheets через Apps Script.

## Запуск

Открой `index.html` в браузере или включи GitHub Pages для репозитория.

## Подключение Google Sheets

1. Открой Google Таблицу.
2. Расширения → Apps Script.
3. Скопируй код из `apps-script/Code.gs`.
4. Опубликуй как Web App.
5. В `app.js` вставь URL Web App в переменную `SHEETS_API_URL`.

Пока URL не указан, приложение работает на демо-данных.
